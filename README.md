## ğŸŒ Connecting to Your Database
1. **Launch sqltools_mcp.**
2. **Select Your Database Type:** Choose MySQL, PostgreSQL, MSSQL, DM8, or SQLite from the dropdown menu.
3. **Enter Connection Details:**
   - **Host:** The address of your database server (for local databases, use `localhost`).
   - **Port:** The port number your database uses (default ports are usually 3306 for MySQL, 5432 for PostgreSQL, etc.).
   - **Username:** Your database username.
   - **Password:** Your database password.
   - **Database Name:** The name of the database you want to connect to.
4. **Connect:** Click on the connect button. If details are correct, you will see your database displayed in the app.
